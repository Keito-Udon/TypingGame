# TypingGame - TOEIC英単語タイピングゲーム

## 概要

TypingGameは，制限時間内にTOEIC頻出英単語をキーボード入力し，スコアを競うWebゲームである．Vanilla JavaScriptで実装されたフロントエンド，Node.js Expressバックエンド，PostgreSQLデータベースで構成される．

## フロントエンド実装詳細

### ファイル説明

#### index.html

HTMLは3つのゲーム画面を定義する．

1. **スタート画面**（start-screen）
   - ゲームタイトル
   - スタートボタン

2. **ゲーム画面**（playing-screen）
   - タイマー表示（残り秒数）
   - 出題単語表示
   - ユーザー入力欄
   - スコア表示

3. **リザルト画面**（result-screen）
   - ゲーム終了メッセージ
   - 最終スコア
   - もう一度ボタン

#### style.css

ゲームの視覚的デザインを定義する．主な要素は以下である．

- **全体レイアウト**：グレー背景に中央配置
- **各画面コンテナ**：白背景，ボックスシャドウ，丸角
- **タイマー**：大きく赤色で表示（48px，＃ff6b6b）
- **単語表示**：太字で大きく表示（36px）
- **入力欄**：18px字体，幅80%，最大300px
- **ボタン**：緑色背景，ホバー時に青に変更
- **スコア**：20px字体で表示

#### app.js

ゲーム全体のロジックを実装するJavaScriptファイルである．

**グローバル変数**
- `words`：TOEIC頻出英単語20語のリスト
- `score`：現在のスコア（正解数）
- `timeLeft`：残り時間（秒）
- `timerInterval`：setIntervalで返されるタイマーID
- `currentWord`：現在出題されている単語

**主要関数**

1. **showScreen(screenId)**
   - 指定されたゲーム画面を表示し，他の画面を隠す
   - 画面遷移を管理する

2. **showNextWord()**
   - 単語リストからランダムに単語を選択
   - 選んだ単語を画面に表示
   - 入力欄をクリア

3. **tick()**
   - 1秒ごとに実行されるタイマー処理
   - 残り時間を1減らす
   - 時間が0以下になったらゲーム終了

4. **startGame()**
   - スコアをリセット（0に初期化）
   - タイマーをリセット（60秒に設定）
   - ゲーム画面に切り替え
   - 最初の単語を表示
   - setIntervalでtick()を1秒ごとに実行開始
   - 入力欄にフォーカスを当てる

5. **endGame()**
   - タイマーを停止（clearInterval）
   - 最終スコアをリザルト画面に表示
   - リザルト画面に切り替え

**イベントハンドラ**

1. **スタートボタン（click）**
   - クリック時にstartGame()を実行

2. **もう一度ボタン（click）**
   - クリック時にstartGame()を実行
   - スタート画面に戻る

3. **入力欄（input）**
   - 入力欄の内容が変わるたびに正誤判定
   - ユーザーの入力が正解単語と一致したら：
     - スコアを1増やす
     - 画面のスコア表示を更新
     - 次の単語を表示

4. **Escapeキー（keydown）**
   - ゲーム中にEscapeキーが押されたら
   - ゲームを強制終了
   - リザルト画面に遷移

## ゲームの動作フロー

1. ユーザーが「Start」ボタンをクリック
2. スタート画面からゲーム画面に遷移
3. タイマーが60から1秒ごとにカウントダウン開始
4. 英単語が画面に表示される
5. ユーザーが単語をキーボードで入力
6. 入力内容が正解単語と一致したら：
   - スコアが1増える
   - 新しい単語が表示される
7. 入力が正解でなければ何もしない（次の入力を待つ）
8. タイマーが0になったらゲーム終了
9. リザルト画面に遷移し，最終スコアを表示
10. ユーザーが「Retry」ボタンをクリックしてスタート画面に戻る

## セットアップと実行方法

1. `/home/yorkshin06/k.maeno/TypingGame/frontend/` に移動

## 実装の技術的特徴

### JavaScript標準機能の活用

- **ES6+構文**：const/let，アロー関数，async/awaitを使用
- **DOM操作**：getElementById，textContent，style.displayで画面操作
- **イベント管理**：addEventListener()で各種イベントをハンドル
- **タイマー管理**：setInterval()とclearInterval()でゲーム時間管理
- **配列操作**：Math.random()とMath.floor()で単語をランダム選択

### セキュリティ考慮事項

現在のフロントエンド実装では，単語データがハードコードされている．
